Pada label pertama, kita akan memberikan label yang digunakan user untuk memilih warna. atribut for dengan value favco merupakan nama yang nantinya akan dipanggil di tag input. pada tag input, kita akan memilih type inputan yang kita inginkan. Disini kita memilih color untuk dapat menginputan jenis warna yang kita inginkan. selanjutnya id dan name kita sesuaikan dengan for yang berada pada label. id dan name sendiri adalah identitas pada label kita.

Pada label pertama, kita akan memberikan label yang digunakan user untuk memilih tanggal. atribut for dengan value b merupakan nama yang nantinya akan dipanggil di tag input. pada tag input, kita akan memilih type inputan yang kita inginkan. Disini kita memilih date untuk dapat menginputan tanggal yang kita inginkan. selanjutnya id dan name kita sesuaikan dengan for yang berada pada label. id dan name sendiri adalah identitas pada label kita.

Pada input selanjutnya type yang kita gunakan adalah button dengan nama CLICK ME!. Ketika button tersebut kita klik, akan muncul pop up pada website kita dengan teks HELLO WORLD!

label myfile merupakan identitas untuk tipe input file yang gunanya untuk memasukan file ke dalam web guna dikirimkan ke http server

type="hidden" menandakan bahwa elemen input ini adalah jenis yang tersembunyi, yang nilainya tidak akan ditampilkan pada web.

id="custId" memberikan identifikasi untuk elemen input sehingga dapat diidentifikasi menggunakan CSS atau JavaScript.

name="custId" menentukan nama yang akan digunakan saat mengirimkan nilai ini bersama dengan data formulir.

value="3487" adalah nilai yang akan dikirimkan bersama dengan data formulir saat formulir tersebut dikirimkan.
